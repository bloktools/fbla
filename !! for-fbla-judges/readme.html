What is Bluehaven?

Bluehaven is a made up city by me and my teammate to give us full creative liberty over the design. This city includes a public high school named after it and a community center located within that just built a new gym/event venue: Bluehaven Community Center Arena. It serves as a placeholder for anywhere a real venue would put its own details like pictures, descriptions, and directions. Speaking of directions, the location selected is St. Petersburg High School (since we can't invent a location in real life).



How to view our project

To view our project, it is necessary to set up a local web server in order for the local references to work. I have included a preset executable that will do all of these steps for you, but in case, for whatever reason, you can't use it, here is how to set it up yourself:

  1. Install some kind of server hosting client, I would recommend Simple Web Server as that is what the instructions moving forward will be based around.
  2. After you set up the program, click "New Server" in the bottom right.
  3. Set the folder path to the main folder. It will not work if you set it to any other folder.
  4. Check the "Accesible on local network" box on.
  5. Click "Create Server."
  6. Now, go to the latest version of Google Chrome and go to 127.0.0.1:8080 into the address bar.
  7. Click enter and you are now on the website.
